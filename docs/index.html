SFuge - Phần mềm hỗ trợ nhập điểm FUGE

Đây là một giải pháp hỗ trợ nhập điểm  nhanh từ bảng Excel vào file FUGE để nộp lên hệ thống.

Phần mềm chạy trên các nền tảng Windows và MacOS

Chỉ với 1 thao tác có thể nhập bảng điểm gần 20 đầu điểm vào file FUGE.

Download phần mềm

Phiên bản dành cho MacOS: https://github.com/snvn1511/release-sfuge/releases/download/MacOS/sfuge-v0.6.app.zip 

Sau khi tải về, giải nén file zip sẽ có file sfuge-v0.6.app
Bạn bấm phải chuột (hoặc bấm giữ touchpad) lên file sẽ có menu Open, chọn Open sẽ chạy được file.

Chạy file này máy tính sẽ hỏi quyền có cho phép mở hay không, bạn chọn "Open Anyway" để mở và bắt đầu sử dụng phần mềm.
Khi phần mềm chạy, hãy kích đúp chuột vào thanh tiêu đề để mở to màn hình làm việc.

Phiên bản dành cho Windows: https://github.com/snvn1511/release-sfuge/releases/download/Windows/SFUGE-Windows-V0.5.zip 

Sau khi tải về, giải nén thư mục là có thể chạy file sfuge.exe để sử dụng.

Các bước thực hiện:

Tải file fg trên trang  https://gv.poly.edu.vn/ 

Bước 1: Chuẩn bị bảng điểm có các tên cột tương ứng theo cấu trúc như sau:

Tiêu đề các cột: Mã SV phải là cột đầu tiên sau đó đến các cột điểm. Tiêu đề các cột điểm thì bấm vào tên lớp trên phần mềm SFUGE sẽ hiện danh sách tên cột, copy từng tên cột trên phần mềm paste vào tiêu đề bảng excel tương ứng (Chú ý copy đúng thì mới nhập được điểm vào phần mềm)

Bước 2: Trên phần mềm chọn tên lớp cần nhập điểm

Bước 3: Copy điểm 

Copy cả bảng điểm bao gồm tiêu đề cột và điểm như ảnh trong bước 1

Bước 4: Paste vào phần mềm và bấm nút Import

Bước 5: Bấm nút Save để lưu thay đổi vào file

Ở bước này, nếu file fg chưa đặt password thì sẽ hiện yêu cầu nhập password, sau khi nhập password xong thì bấm Save lại một lần nữa để lưu file.

 

Chú ý : 

- Khi thao tác phải thật chính xác tên lớp đang được chọn ở bên trái, tránh tình huống SV học nhiều lớp, có thể import nhầm điểm.

 - Dữ liệu điểm phải làm thật chính xác ở bảng Excel trước khi  đưa vào phần mềm.
